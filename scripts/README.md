# __ECKO: Scripts__

## __is-master.sh__
Usage: `is-master`

This script, when run, will `echo "MASTER"` and return `EXIT_SUCCESS` if the node it is run on is designated as the master node (Node to run the control plane on). Otherwise, it will return `EXIT_FAILURE`
